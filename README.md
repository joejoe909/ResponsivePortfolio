# ResponsivePortfolio 
by Joe Farrish.

This is my Responsive Layout page useing HTML, CSS and Bootstrap.

I was able to make use of the bootstrap framework via the documentation. For the Image grid I disected the Albums example that comes with the bootstrap framework when you download it. https://getbootstrap.com/docs/4.5/examples/album/ this served as the bases for my own portfolio.

Each page index.html, contact.html and portfolio.html each utilizes it's own respective css file index_style.css, contact_style.css and portfolio_style.css
